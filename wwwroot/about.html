A-propos d'Innovisite

Innovisite est un sîte web permettant d'accéder à des données géroréférencées provenant des sources publiques des institutions Françaises et Européennes. C'est une initiative personnelle qui a pour but de faciliter l'accès au données publiques au plus grand nombre.

L'architecture du sîte Innovisite est ouverte et l'ensemble du code source est disponible sur github. Je tiens à remercier tout particulièrement NICTA, les différents contributeurs de TerriaJS, NationalMap et les créateurs de Cesium pour avoir mis à disposition les codes sources sous license Apache Version 2.0.

Innovisite

  • fournit un accès simple aux données spatiales des institutions publiques
  • permet de retrouver l'information à différentes échelles ; national, régional, départemental etc.
  • fournit une base de données commune permettant d'indexer les services de données géoréférencés qu'ils soient issus de la communauté ou bien commerciaux.

Pour savoir ce qui est disponible, visitez le Catalogue de Données sur le sîte en ligne Innovisite. Cliquez sur la tabulation Catalogue de Données et parcourez les différents sous ensembles en fonction de leur noms.

Comment utiliser Innovisite

Allez à l'addresse http://www.innovisite.com/. L'interface vous pouvez consulter la documentation sur le lien Aide & FAQ qui est également présent dans la barre de navigation de cette page.

Pour plus d'informations écrivez-nous à l'addresse email contact@innovisite.com. Si vous constatez des bugs n'hésitez pas à nous en faire part.

Logiciel Innovisite

Innovisite est distribué en Open Source et disponible comme projet Github. Les plateformes initiales, nationalmap et TerriaJS, ont été développées par NICTA.

Le code est sous license Apache Version 2.0 (la "License"). Vous ne pouvez utiliser les fichiers source sinon en étant compatible avec la License. Une copie de la License est disponible sur http://www.apache.org/licenses/LICENSE-2.0.

Autres logiciels Open Source utilisés

Pour développer Innovisite, les logiciels principaux suivants ont été utilisés:

  • nationalmap
  • TerriaJS
  • Cesium
  • Leaflet
  • La liste complète est disponible avec le code source Innovisite

Utilisations des données OpenData

Vous pouvez utiliser les données issues des sources OpenData en vous informant sur la license qui y est attribuée.

Vous disposerez toujours d'un bouton d'information vous permettant de retrouver le flux d'origine.

Traitement des données:

Les informations issues des sources OpenData ne sont pas post traitées excepté lorsqu'un filtre est explicitement activé dans la vue navigation.
