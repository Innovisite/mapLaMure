Innovisite France version Beta

Innovisite est une plateforme Web ouverte permettant d'accéder à des données géroréférencées provenant des sources publiques et des institutions Françaises et Européennes. C'est une initiative personnelle (citoyenne) qui a pour but de faciliter l'accès au données publiques au plus grand nombre.

Les principales fonctionalités de cette plateforme sont:

  • fournir un accès simple aux données spatiales des institutions publiques
  • permettre de retrouver l'information à différentes échelles ; national, régional, départemental etc.
  • fournir une base de données commune permettant d'indexer les services de données géoréférencés.

Pour savoir ce qui est disponible, visitez le Catalogue de Données sur le sîte en ligne Innovisite. Cliquez sur la tabulation Catalogue de Données et parcourez les différents sous ensembles en fonction de leur noms.

Utilisations des données OpenData

Vous pouvez utiliser les données issues des sources OpenData en vous informant sur la license qui y est attribuée.

Vous disposerez toujours d'un bouton d'information vous permettant de retrouver le flux d'origine.

Traitement des données:

Les informations issues des sources OpenData ne sont pas post traitées excepté lorsqu'un filtre est explicitement activé dans la vue navigation.

Si vous voulez effectuer des traitements spécifiques sur les données, contactez-moi !
